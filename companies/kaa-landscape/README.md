# KAA Landscape

Landscape architecture agent company for Paperclip. Automates client intake, project management, design research, compliance review, and client communications for a SF Bay Area firm with a 10,000+ project archive.

## How it works

KAA operates as a **hub-and-spoke** organization. The CEO triages work across four service tiers and delegates to specialists who report back independently.

## Org Chart

| Agent | Title | Reports To | Focus |
|-------|-------|------------|-------|
| CEO | Chief Executive Officer | — | Strategy, intake triage, cross-team coordination |
| Project Manager | Project Manager | CEO | PRD management, milestones, team coordination |
| Design Archivist | Design Research Specialist | Project Manager | Archive search, plant palettes, precedent research |
| QA Reviewer | Plan Compliance Reviewer | Project Manager | Building codes, ADA, MWELO, budget validation |
| Client Comms | Client Communications Specialist | Project Manager | Proposals, updates, meeting prep, milestone notifications |

## Service Tiers

- **Concept** ($299) — Initial design concept and plant palette
- **Builder** — Full plans for contractor handoff
- **Concierge** — Design + project management through installation
- **White Glove** — End-to-end design, installation oversight, and maintenance planning

## Getting Started

```bash
paperclipai company import --from companies/kaa-landscape/
```
