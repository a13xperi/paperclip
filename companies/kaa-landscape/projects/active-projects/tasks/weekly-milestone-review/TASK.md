Weekly audit of all active KAA engagements:

1. **Pull Active Projects** — List all projects with status In Progress
2. **Check Milestones** — Compare current date against each project's milestone schedule
3. **Update Status** — Mark each project as On-Track / At-Risk / Blocked with rationale
4. **Flag Overdue** — Identify milestones past their target date without completion
5. **Escalate Blockers** — Surface Blocked projects to CEO with context and recommended action
6. **Post Report** — Publish weekly status summary as a task comment
