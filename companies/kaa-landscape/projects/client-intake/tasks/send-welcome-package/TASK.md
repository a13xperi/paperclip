Prepare and send the client welcome package:

1. **Select Tier Template** — Use the appropriate welcome template for Concept / Builder / Concierge / White Glove
2. **Personalize Content** — Include project name, assigned PM, and key dates from the PRD
3. **Attach Onboarding Checklist** — Tier-specific list of what the client needs to provide (site access, HOA docs, inspiration photos, etc.)
4. **Set Expectations** — Communication cadence, response SLAs, and next steps
5. **Send & Log** — Deliver via email and log send confirmation to the project record
