Build a structured PRD from the triaged intake data:

1. **Client Profile** — Contact info, service tier, communication preferences
2. **Site Conditions** — Location, climate zone, lot size, sun exposure, soil type, existing structures
3. **Design Brief** — Client vision, style references, functional requirements, constraints
4. **Milestones** — Draft timeline with key gates (concept approval, permit submission, planting schedule)
5. **Budget** — Tier-appropriate budget framework with contingency ranges
6. **Create in Notion** — Publish PRD to the KAA project database and link to client record
